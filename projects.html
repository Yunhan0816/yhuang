Inhaled

Developed at BostonHacks 2019, Inhaled allows asthma patients and their doctors to track their medication and respiratory well-being, providing the ability to create the most effective regime for each patient.

Frontend is built with Vue.js and backend is built with firebase. Weather and Gmap API are integrated in weather view for patients to refer and determine the safety to go out. View more on Devpost

Song Lyrics Analysis

Sentiment analysis of my favorite bands' lyrics. Pulled and cleaned lyrics data by using Genius.com API and web-scraping with BeautifulSoup.

Used wordcloud library in Python to generate word cloud for each band and used NLTK's sentiment analyzer to generate normalized unidimensional metric of sentiment, finding the positivity, negativity, and neutrality of the lyrics by each artist, and illustrated the result by plotting the graph.

Ameelio Connect

Ameelio Connect is a website built for inmates and regular users (their family and friends) the access to have video/message communication, and facility administrators to manage the connections. Users can sign up, add inmates, schedule to connect with inmates and video chat when their request has been approved by an admin.

Used Vue.js as frontend and firebase as backend.
